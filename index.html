Git-GitHub

  • create a new repository on the command line - создайте новый репозиторий в командной строке

    • git init
    • git add README.md
    • git commit -m "first commit"
    • git remote add origin https://github.com/vintsan/Git-GitHub.git
    • git push -u origin main

  • or push an existing repository from the command line - или запустите существующий репозиторий из командной строки

    • git remote add origin https://github.com/vintsan/Git-GitHub.git
    • git branch -M main
    • git push -u origin main

Коммиты

  • git status
  • git add . - добавляет все файлы в stage
  • git commit -m "comment" - commit = запись
  • git log / git log --oneline - инфа по коммитам
  • git remote -v - информация о репозитории (ссылка на наш репозиторий)
  • git push [rep_link] [branch_name] - например [origin - ссылка на наш репозиторий] [master - название ветки]

Ветки в Git. Создание веток и их отправка на GitHub

  • git branch - просмотр созданных веток
  • git branch develop - создание ветки "develop"
  • git checkout develop - переход на ветку "develop"
  • git remote -v - информация о репозитории (ссылка на наш репозиторий)
  • git push [rep_link] [branch_name] - сохранение на github (например [origin - ссылка на наш репозиторий] [master - название ветки])
